## WARIOWARE PROYECT IN C

So, this is a WarioWare-style game that I made because I really love
WarioWare and I decided to make the game just because

## WHAT DOES VERSION 1.0 INCLUDE?

- An ASCII menu with logos where you can play the game, see the credits of the project and exit the game
- A total of 8 unique ASCII microgames (7 of them related to the programming field) with terminal-like
inteface
- Every microgame has instructions for how to play it and the amount of time you have to complete the
microgame

## WHAT WILL VERSION 2.0 INCLUDE?

- Music for the menu, microgames and the microgame itself
- Some microgames might receive upgrades
- Maybe 2 or 3 more microgames
- More clean, structured code

### NOTES:
